## 2. 详细描述（Detailed Description）

### 扩展介绍

AI Bookmarks 是一款智能书签管理扩展，利用先进的 AI 技术自动为您的书签进行分类和标签管理。只需保存书签，扩展会自动分析网页内容，提取关键信息，并智能生成 2-5 个精准的分类标签，自动将书签整理到对应的文件夹中。

告别手动整理书签的繁琐，让 AI 帮您构建一个井然有序的书签库。无论是技术文章、新闻资讯、购物网站还是学习资料，AI Bookmarks 都能准确识别内容类型，为您自动归类。

通过 Cloudflare Workers 提供的云端服务，您的书签数据安全可靠，AI 分析快速高效。扩展支持多语言标签生成，自动适配您的浏览器语言环境。

### 主要功能

- **AI 智能分类**：基于 Cloudflare Workers AI 的先进语言模型，自动分析网页内容并生成精准标签
- **自动提取元数据**：智能提取网页标题、描述、Open Graph 信息和正文内容
- **多标签管理**：每个书签可获得 2-5 个相关标签，支持多维度分类
- **自动文件夹组织**：根据主标签自动创建文件夹并移动书签，保持书签栏整洁
- **多语言支持**：标签自动适配浏览器语言（支持中文、英文、日文、韩文等多种语言）
- **云端同步**：通过 Cloudflare Workers 后端实现数据同步和 AI 处理
- **实时通知**：书签保存后即时显示生成的标签，让您随时了解分类结果
- **隐私保护**：仅处理书签 URL 和元数据，不收集个人浏览历史

### 使用场景

- **技术开发者**：自动整理技术文档、开源项目、技术博客
- **内容创作者**：分类管理灵感来源、参考资料、行业资讯
- **学生研究者**：组织学习资料、论文文献、课程资源
- **日常用户**：管理购物网站、新闻资讯、生活服务等各类网站

### 技术特点

- 基于 Chrome Manifest V3 标准开发，安全可靠
- 使用 TypeScript 构建，代码质量高
- 采用 Cloudflare Workers 边缘计算，响应速度快
- 集成 Cloudflare Workers AI，AI 处理能力强大
- 支持 D1 数据库存储，数据持久化可靠

## 4. 使用方法（How to Use）

### 第一步：安装配置
安装扩展后，点击扩展图标，在设置页面配置您的 Cloudflare Worker URL 和 API Key。

### 第二步：保存书签
像往常一样保存书签到"其他书签"文件夹根目录，扩展会自动检测新书签。

### 第三步：AI 自动处理
扩展自动提取网页内容，调用 AI 生成标签，并显示通知告知分类结果。

### 第四步：自动整理完成
书签自动移动到对应的标签文件夹中，您的书签库保持整洁有序。

## 5. 隐私说明（Privacy）

### 数据收集和使用

AI Bookmarks 高度重视您的隐私安全，我们承诺：

- **仅处理书签数据**：扩展仅访问您主动保存的书签 URL、标题和网页元数据
- **不收集浏览历史**：不追踪或记录您的浏览行为和访问历史
- **不存储个人信息**：不收集任何个人身份信息（姓名、邮箱、地址等）
- **数据传输加密**：所有数据通过 HTTPS 加密传输到您配置的 Cloudflare Worker
- **用户完全控制**：您可以随时删除扩展和相关数据，所有本地数据将被清除

### 权限使用说明

扩展请求的权限及用途：

- **bookmarks（书签）**：读取和组织您的书签，创建文件夹并移动书签
- **storage（存储）**：保存扩展配置（Worker URL、API Key）和最近处理的书签信息
- **notifications（通知）**：在书签处理完成后显示分类结果通知

### 第三方服务

- **Cloudflare Workers**：用于运行 AI 分析服务，处理书签元数据并生成标签
- **网页内容获取**：扩展会访问书签对应的网页以提取元数据（标题、描述等）

我们不会将您的数据分享给任何第三方，也不会用于广告或营销目的。
